To Do List

Crypto Trader

Aplicación web que utiliza la API de CryptoCompare para obtener el precio actual, el precio más alto y más bajo del día, la variación en las últimas 24 horas y la última actualización de las 10 criptomonedas mas buscadas en el mercado.

Calculator

Calculadora básica utilizando tres archivos de JavaScript: uno para el display, otro para la funcionalidad de la calculadora y uno para los botones.

Rock Paper Scissors

Este código implementa un juego básico de piedra, papel o tijeras en JavaScript, donde el usuario juega contra la computadora. El código genera una selección aleatoria para la computadora y luego determina quién ganó. El puntaje del usuario y la computadora se actualizan y se muestran en la pantalla.

Weather App

Js con API de OpenWeatherMap para obtener el clima de una ciudad ingresada por el usuario. Muestra el clima en Celsius, temperatura máxima y mínima, y muestra un mensaje de error si la ciudad no se encuentra.

Responsive Contact Form
